HTML

  • The <!DOCTYPE html> declaration lets the browser know to expect HTML on the webpage.
  • The <head> element contains information, or metadata, about the webpage.
  • The <body> element represents the visible content shown to the user.
  • Elements usually have a start tag, some content, and a closing tag.
  • Tags should be nested in correct inverse order... the last tag opened must be closed first.
    Example: <p><em> --These tags are nested in correct order-- </em></p>
  • Attributes provide additional properties about elements and are specified in the start tag.
    Example: <tagname attribute="value"></tagname>
  • Semantic elements clearly describe meaning to both the browser and the developer.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the inside of an element
  • A CSS rule can designate a class attribute as the selector
  • A period preceding [card] designates this as a class selector. Any HTML element assigned to the class, [card], will gain these property assignments from the CSS.
  • External style sheet links are placed in the head section, where metadata is stored

Git

  • git pull origin main: pulls and syncs the remote base branch (github) into your local branch
  • git status: check working branch
  • git branch: list branches. * = current active branch
  • git checkout main: switch to local main branch
  • git checkout branch-name: switch branch
  • git checkout -b branch-name: creates new branch and switches to it
  • git branch -d branch-name: deletes branch only if it has been fully merged with upstream branch. Use -D to force delete irrespective of its merged status

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A variable must be defined first before another variable makes reference to it.
  • Use / / --for single line comments--
  • Use / * --for multi line comments-- * /
  • JS code can be added directly to an HTML document or by linking to an external file.
