  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the inside of an element
  • A CSS rule can designate a class attribute as the selector
  • A period preceding [card] designates this as a class selector. Any HTML element assigned to the class, [card], will gain these property assignments from the CSS.
  • External style sheet links are placed in the head section, where metadata is stored

Git

  • git pull origin main: pulls and syncs the remote base branch (github) into your local branch
  • git status: check working branch
  • git branch: list branches. * = current active branch
  • git checkout main: switch to local main branch
  • git checkout branch-name: switch branch
  • git checkout -b branch-name: creates new branch and switches to it
  • git checkout -d branch-name: deletes local branch
